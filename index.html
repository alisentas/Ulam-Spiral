Ulam Spiral by Ali Şentaş

Usage

Cube width is width of every colored prime number cubes withd, a value between 1-10 is recommenden.
Canvas width and height are obviously canvas size variables
Count is the how many numbers should algorithm check for prime numbers, for example: if its 100, program will fill numbers between 1-100, not 100 prime numbers, beware huge numbers bigger than 5,000,000 etc.
Color bust be a valid color, like black, red, blue etc. You can see which colors are available in this page
Start number is the number which my algorithm should start drawing, 1 or 41 is recommended
